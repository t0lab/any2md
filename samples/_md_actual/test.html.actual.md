# Telerik UI for ASP.NET MVC Chart

## Description

The [Telerik UI for ASP.NET MVC chart](https://www.telerik.com/aspnet-mvc/charts)
is a data visualization component, which allows you to
graphically represent your data. It is useful when you would like to
utilize modern browser technologies such as
[SVG](https://en.wikipedia.org/wiki/Scalable_Vector_Graphics)
or
[Canvas](https://en.wikipedia.org/wiki/Canvas_element)
(with
a fallback to
[VML](https://en.wikipedia.org/wiki/Vector_Markup_Language)
for older IE versions) for interactive data visualizations.

The component offers a variety of chart types such as area, bar, line,
scatter, polar, radar, pie and donut, stock (OHLC) and many more.

A Bar chart displays data as horizontal bars whose lengths vary
according to their value. You can use a Bar chart to show a comparison
between several sets of data (for example, summaries of the unique and
total site visitors for certain period).
The chart can fetch data for its series from either local or remote data
source. You can find more information about the MVC chart in the
[product documentation](https://docs.telerik.com/aspnet-mvc/helpers/chart/overview).

**Key Features of Telerik MVC Charts**

- Various Chart Types
- A Number of Data Types
- High Performance
- Export to PDF
- Simple-to-Customize Look and Feel
- Rich Server and Client-side API
- Server and Client-side Data Binding
- Built-in Themes
- Intuitive Built-in Animations
- Multiple Series and Combination Charts
- Plot Bands
- Rich Cross-Browser and Mobile Support

More about [Telerik MVC Charts](https://www.telerik.com/aspnet-mvc/charts).
